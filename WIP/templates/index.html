Refugee Movement Overtime

Erin, Sruthi, JD, Megan, Sirishk

Overview

Project Description

The UN estimates 65.6 million people displaced globally by conflict. Since 1951, the UN has kept statistics on refugee movement. Using a UN-refugee dataset from dataword that will be stored in a SQL database.

Goals

We want to visualize the movement of refugees across specific regions by using an interactive chart. The chart will display human migration.

Project Details

  • Heatmap (of refugees to/from country)
  • Slide date function for years
  • Flight Path Map
  • Adipiscing Elit

Tools

  • Chartjs
  • Amcharts
